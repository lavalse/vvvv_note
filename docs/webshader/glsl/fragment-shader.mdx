```glsl
float strength = mod(vUv.y * 5.0, 1.0);
gl_FragColor = vec4(vec3(strength), 1.0);
```
mod means modulo 就是求小数部分，类似于js里面的 %

```glsl
float strength = mod(vUv.y * 5.0, 1.0);
strength = step(strength, 0.5);
gl_FragColor = vec4(vec3(strength), 1.0);
```
step可以用来判断小于0.5则为0，大于0.5则为1.
